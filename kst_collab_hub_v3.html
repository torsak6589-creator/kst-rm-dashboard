KST Collaborative Hub
PRODUCTION SETUP — FIREBASE REALTIME DB
ตั้งค่าครั้งเดียว ใช้ได้ตลอด — ใช้เวลา ~10 นาที
1
สร้าง Firebase Project
ไปที่ console.firebase.google.com → Add project → ชื่อ "kst-collab" → ไม่ต้อง Google Analytics
🔗 เปิด Firebase Console
2
เปิด Realtime Database
Build → Realtime Database → Create database → เลือก Southeast Asia → Start in test mode (30 วัน จากนั้นตั้ง Rules)
3
ตั้ง Security Rules (หลัง 30 วัน)
Realtime Database → Rules → วางโค้ดนี้:
{ "rules": { "kst": { ".read": true, ".write": true } } }
* สำหรับใช้ภายใน — ปลอดภัยพอสำหรับ internal tool
4
คัดลอก Firebase Config
Project Settings (⚙️) → Your apps → Web app → Register app → คัดลอก firebaseConfig
วาง Firebase Config ที่นี่
Config บันทึกใน localStorage ของ browser นี้เท่านั้น · ไม่มีการส่งออก
