📋 วิธีตั้งค่า Azure App Registration (ทำครั้งเดียว)
1. ไปที่ portal.azure.com → App registrations → New registration
2. ตั้งชื่อ: KST-RM-Dashboard · Supported account types: Single tenant
3. Redirect URI: Single-page application (SPA) → ใส่ URL ที่จะ host ไฟล์นี้
4. หลัง register → คัดลอก Application (client) ID
5. ไปที่ API permissions → Add: Files.Read, Sites.Read.All, User.Read → Grant admin consent
6. แก้ CLIENT_ID = 'YOUR_AZURE_CLIENT_ID' ในไฟล์นี้
